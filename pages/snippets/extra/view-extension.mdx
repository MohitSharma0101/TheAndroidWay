## Resize the View

Set Width and height of a View Programmatically.

### UsageğŸ¯
Call it on any `View`
```
myView.resize(50 , 100)
```

### CodeğŸ’»

```
fun View.resize(width: Int, height: Int) {
    val lp = this.layoutParams
    lp?.let {
        lp.width = width
        lp.height = height
        layoutParams = lp
    }
}
```
