<Anchor>Resize the View</Anchor>

Set Width and height of a View Programmatically.

### UsageðŸŽ¯
Call it on any `View`
```javascript

```

### CodeðŸ’»

```javascript
fun View.resize(width: Int, height: Int) {
    val lp = this.layoutParams
    lp?.let {
        lp.width = width
        lp.height = height
        layoutParams = lp
    }
}
```
